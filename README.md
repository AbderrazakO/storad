## Overview

Storad is an ecomerce website...

\
Technologies used :

- ReactJs
- Sass
- React Router
- RXJS

## Setup

Clone the project

```bash
git clone https://github.com/AbderrazakO/storad.git
```

Install dependencies:

```bash
npm i
```

Start the server:

```bash
npm start
```
